Chaque jour de bourse, le bot enregistre tous les tickers scannés puis, pour les setups, trois groupes comparés : A = direction seule, B = direction + timing IA, C = contrôle aléatoire. Les résultats sont capturés à +1, +3 et +5 séances avec les vrais prix. Verdict quand on atteindra 50+ signaux : si B bat A après frais, l'IA a une valeur. Rien n'est retouché, les échecs de données sont comptés.
